# Exercise 4: Functions

## Objective
To write PL/SQL functions to perform calculations and return values to calling environments.

## Functions
1. **`CalculateAge`**: Input: Customer's DOB. Output: Age in years.
2. **`CalculateMonthlyInstallment`**: Input: Loan amount, annual interest rate, duration (in years). Output: Monthly EMI amount (calculated using compound amortization).
3. **`HasSufficientBalance`**: Input: Account ID, check amount. Output: `'TRUE'`, `'FALSE'`, or `'INVALID_ACCOUNT'`.
