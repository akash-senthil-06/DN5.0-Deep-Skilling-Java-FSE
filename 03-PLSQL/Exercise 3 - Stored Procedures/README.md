# Exercise 3: Stored Procedures

## Objective
To modularize database queries and business execution logic into reusable SQL Stored Procedures.

## Stored Procedures
1. **`ProcessMonthlyInterest`**: Runs a bulk update applying a 1% balance increase on all accounts of type `'Savings'`.
2. **`UpdateEmployeeBonus`**: Updates salaries for a specified department by adding a user-defined percentage bonus.
3. **`TransferFunds`**: Transfers funds from one account to another, checking for sufficient balance and issuing warnings if the check fails.
